# Parkly — Sydney parking, live

A web app that shows real-time parking availability across Sydney, ranked by availability, commute time, walk distance, and price.

## Features
- Live-updating parking spot availability
- Destinations: Sydney CBD, Bondi Beach, SCG / Moore Park, Chatswood
- Filters: arrival time, day of week, weather, events, premium mode
- Interactive Leaflet map with colour-coded markers

## Stack
- Vanilla HTML/CSS/JS (prototype)
- [Leaflet.js](https://leafletjs.com/) for maps
- [CartoDB](https://carto.com/) tile layer

## Status
Currently using simulated data. Real data integration coming next.

## Getting started
Open `parkly.html` in a browser — no build step needed.
